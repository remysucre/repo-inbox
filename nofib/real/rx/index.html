RX: an interpreter for Rational Tree Languages

RX is a system that knows about regular tree grammars, finite tree automata, and rational tree expressions.

You may feed some regular tree grammars to my RX interpreter and do some computations. See the online examples, and edit them to taste.

If you want to use RX for real work, please get the latest source release and install it on your system. The sources are written in the pure lazy functional programming language Haskell and can be compiled with ghc or hbc, or interpreted under hugs.

Please tell me if you built RX, and have any comments or if you just want to be informed on further releases.
